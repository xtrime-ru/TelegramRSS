Hello!

This is my telegram rss & json server. It can return 10 last posts from any public Telegram channel as RSS or JSON.

Features

  • Fast async swoole server;
  • Get any public telegram posts from groups as json or RSS;
  • Full media support. Access any media from messages via direct links.

Examples

  • JSON: https://tg.i-c-a.su/json/breakingmash;
  • RSS: https://tg.i-c-a.su/rss/breakingmash;
  • Custom posts count: https://tg.i-c-a.su/rss/breakingmash?limit=100;
    Maximum limit - 100 posts
  • Pagination: https://tg.i-c-a.su/rss/breakingmash/2;
  • Media preview: https://tg.i-c-a.su/media/breakingmash/10738/preview;
  • Media full: https://tg.i-c-a.su/media/breakingmash/10738.

Source Code

This service is combination of two my micro-services:

  • TelegramSwooleClient: https://github.com/xtrime-ru/TelegramSwooleClient;
  • TelegramRSS: https://github.com/xtrime-ru/TelegramRSS.

Limits and restrictions

  • Only public channels;
  • RPM Limit: 15 requests per minute;
  • Temporary ban on any error, for example: invalid channel name or private channel.
